# 八字小程序页面测试指南

## 快速开始

### 1. 启动本地服务器

由于浏览器的CORS限制，需要通过HTTP服务器访问页面：

```bash
# 进入项目目录
cd /Users/zdxy/codes/bazi-mini-programme/bazi-front-design/design-20260406

# 启动Python HTTP服务器
python3 -m http.server 8000

# 或者使用Node.js的http-server
npx http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000/src/pages/home/home.html`

## 已知问题

由于使用Mock数据，以下功能为模拟实现：

1. **轮询机制**：目前是随机模拟，约30%概率完成计算
2. **表单提交**：创建档案和匹配实际上没有保存到后端
3. **页面跳转**：使用`window.location.href`直接跳转
4. **状态持久化**：删除档案只在当前会话有效
