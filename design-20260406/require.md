# 需求文档
> 本文档说明：本文档会被CLAUDE.md所引用，用于动态加载整体需求。

## 项目说明
本项目是一个八字五行相关的web项目，本目录下需要做的是前端页面的设计，本目录不是完整的前端实现，而是前端ui的设计稿，需要做css、html、js的一些代码实现，用于快速实现与验证设计。

## 项目规范
- 本项目需要维护一个结构文档 [结构文档](./structure.md) 用于说明页面的层级结构与交互顺序，每次生成新的页面或者组件都要在这个文档中进行更新。
- 组件与元素之间的样式需要尽可能地统一，公共的一些样式需要抽取出来做成公共的，样式放在[样式目录](./style/)下面。
- 在进行设计的时候需要有层级的概念，比如先设计背景再设计上面的组件，当用户说明需要微调某个组件的时候不要去动背景。在进行设计的时候需要先设计组件再将组件整合起来，不要一上来就生成整个页面，组件的设计可以放在[组件目录](./components/)下，如果单个组件需要多个文件可以新建子目录，（组件需要尽可能少，尽量复用）。
- 需要的mock数据可以以json的形式放在[mock数据目录](./mock-data/)目录下，数据不要硬编码在html、js中，方便维护。
- 如果在和用户交流中发现了新的需求你觉得需要持久化下来，需要补充到本文档中。
- 哪些实现细节不清楚的需要问用户，**不要自由发挥**， 了解了之后记录到require.md文件中，也即本文件。
- 在不明白一些五行命理相关知识的时候可以先在本目录或者上层目录搜一下，之前做过调研以及一些demo的代码。

## 整体需求
- **风格** UI需要现代简洁，兼顾主题的神秘感。可以参考 /ultimate-frontend-design 技能中的一些规范。
- **形态与主题** 移动端、PCweb 形态，暗黑主题与明亮主题，设计的时候需要兼顾，需要特别注意的如下：
  - 在PC web端的时候有些交互，比如hover这样的交互在移动端是不存在的，在移动端的交互比如拖拉等等在PC端不太好用，有些交互需要单独设计以适配不同的场景；
  - 在暗黑主题与明亮主题，文字与标识要注意跟随主题变化，易读是第一位的。
- **主题颜色** 五行颜色（暗黑模式）白绿蓝红黄 ，（亮色模式）黑绿蓝红黄
- 之前已经设计了一版主题，可以通过 [前版本设计文档](../glow-glass-design/design-thesis.md)来进行索引与参考。


## 细节需求
每个用户可以新建N个八字档案，用户将某个档案设置为当前档案时所有的其他操作都是针对该档案（目前暂定最高10个档案），数据上保证不会出现超过十个档案的情况。

组件的包含关系会用大括号层级的方式表达，如果一个组件内部是一个数组则只会在第一个元素中详细列出对应的内部元素，后续的将进行简略示意而不包含内部细节。每个页面将使用先描述有哪些内容再描述用户对于特定组件或页面的具体需求。

每个推荐约定小于10个结果

三种情况：

```
{
    翻页容器{
        推荐八字的 八字与命卦summery card 1
        推荐八字的 八字与命卦summery card 2

        段落1 card
```

## 组件索引

所有组件位于 `components/` 目录，每个组件包含 `.html`（示例）、`.css`（样式）、`.js`（逻辑）

### 基础组件
- **background** - 背景动画（五行光晕旋转 + 边缘高光）
- **card** - 液态玻璃卡片（光晕跟随 + 磨砂玻璃 + 颜色主题）
- **button** - 按钮（主题色光晕 + 悬停效果）
- **input** - 输入框（边框光晕 + 主题适配）
- **selector** - 选择器（下拉菜单 + 主题色）
- **loading-spin** - 短时加载动画（旋转光点）

### 功能组件
- **swiper** - 翻页容器（PC端3D Coverflow + 移动端滑动）
- **menu** - 环形菜单按钮（底部展开导航）
- **processing** - 长时处理动画（太极旋转 + 辉光）
- **markdown** - 增强解析器（Marked.js + Highlight.js + Mermaid + 五行主题色）
- **wuxing-chart** - 五行生克图（SVG动画 + 节点飘动 + 日主星星公转）

### 图标组件
- **icons** - SVG图标库（五行图标 + 其他图标）

### 使用方式

**CSS 变量引用**（推荐）：
```css
.card[data-theme="wood"] {
  --theme-hue: 142;
  --theme-saturation: 74;
  --theme-lightness: 65;
}
```

**JavaScript 动态设置**：
```javascript
card.style.setProperty('--theme-hue', hue);
card.style.setProperty('--theme-saturation', saturation);
card.style.setProperty('--theme-lightness', lightness);
```

**完整颜色系统**见 `style/tokens.css`
