使用说明

基础用法

<div id="markdown-content" class="markdown-body"></div>

<script>
  const markdownText = '# 标题\n\n这是**粗体**文字。';
  const html = marked.parse(markdownText);
  document.getElementById('markdown-content').innerHTML = html;
</script>

组件特性

  • ✅ 使用Marked.js解析Markdown
  • ✅ 代码语法高亮（Highlight.js）
  • ✅ 暗黑/明亮主题适配
  • ✅ 玻璃质感卡片样式
  • ✅ 响应式设计
  • ✅ 支持所有标准Markdown语法
  • ✅ 自定义样式与设计系统统一
  • ✅ 表格、列表、引用等元素美化

支持的Markdown语法

  • 标题（H1-H6）
  • 粗体、斜体、删除线
  • 有序列表、无序列表
  • 代码块、行内代码
  • 引用块
  • 表格
  • 链接、图片
  • 分割线

依赖库

  • Marked.js - Markdown解析器
  • Highlight.js - 代码高亮
