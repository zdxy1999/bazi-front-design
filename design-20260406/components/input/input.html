使用说明

基础用法

<!-- 基础输入框 -->
<input type="text" class="input" placeholder="请输入内容">

<!-- 带标签 -->
<div class="input-group">
  <label class="input-label">姓名</label>
  <input type="text" class="input" placeholder="请输入姓名">
</div>

<!-- 带图标 -->
<div class="input-wrapper">
  <svg class="input-icon">...</svg>
  <input type="text" class="input" placeholder="请输入用户名">
</div>

组件特性

  • ✅ 液态玻璃质感
  • ✅ 胶囊形状（左右圆润）
  • ✅ 暗黑/明亮主题适配
  • ✅ 多种尺寸：sm、默认、lg
  • ✅ 多种状态：焦点、禁用、错误、成功
  • ✅ 五行主题色：wood、fire、earth、metal、water
  • ✅ 支持图标、前后缀
  • ✅ 完整的表单支持
  • ✅ 键盘导航支持
  • ✅ 可访问性优化

Class列表

基础

  • input - 基础输入框
  • input-group - 输入框组容器
  • input-label - 标签
  • input-wrapper - 带图标的包装器
  • input-icon - 图标
  • input-prefix - 前缀
  • input-suffix - 后缀

尺寸

  • input-sm - 小输入框
  • input-lg - 大输入框

状态

  • input-error - 错误状态
  • input-success - 成功状态
  • input-error-text - 错误提示
  • input-success-text - 成功提示

类型

  • textarea - 多行文本框

主题色

  • input-wood - 木（绿色）
  • input-fire - 火（红色）
  • input-earth - 土（黄色）
  • input-metal - 金（白色）
  • input-water - 水（蓝色）
