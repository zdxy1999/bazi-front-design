使用说明

基础用法

<!-- Radio 单选框 -->
<label class="radio">
  <input type="radio" name="group" value="1" checked>
  <span class="radio-text">选项一</span>
</label>

<!-- Checkbox 复选框 -->
<label class="checkbox">
  <input type="checkbox" name="item" value="1" checked>
  <span class="checkbox-text">选项一</span>
</label>

<!-- Radio Button 按钮式单选 -->
<label class="radio-button">
  <input type="radio" name="group" value="1" checked>
  <span class="radio-button-text">选项一</span>
</label>

<!-- Toggle 开关 -->
<label class="toggle">
  <input type="checkbox" checked>
  <span class="toggle-slider"></span>
  <span class="toggle-text">开启</span>
</label>

五行主题色

<!-- 木（绿色） -->
<label class="radio radio-wood">
  <input type="radio" name="theme" checked>
  <span class="radio-text">木</span>
</label>

<label class="checkbox checkbox-wood">
  <input type="checkbox" checked>
  <span class="checkbox-text">木</span>
</label>

<label class="radio-button radio-button-wood">
  <input type="radio" name="theme-btn" checked>
  <span class="radio-button-text">木</span>
</label>

<label class="toggle toggle-wood">
  <input type="checkbox" checked>
  <span class="toggle-slider"></span>
  <span class="toggle-text">木</span>
</label>

自定义颜色

<!-- 通过 CSS 变量自定义颜色 -->
<style>
:root {
  --accent-primary: rgba(255, 105, 180, 1); /* 粉红色 */
}
</style>

<!-- 或使用内联样式 -->
<label class="radio" style="--accent-primary: rgba(34, 197, 94, 1);">
  <input type="radio" name="custom" value="1">
  <span class="radio-text">绿色选项</span>
</label>

组件特性

  • ✅ 统一选中效果：所有选择器选中时整个内部点亮
  • ✅ 液态玻璃质感
  • ✅ 暗黑/明亮主题适配
  • ✅ 五行主题色：wood、fire、earth、metal、water
  • ✅ 自定义颜色：通过 CSS 变量 --accent-primary
  • ✅ 禁用状态支持
  • ✅ 平滑动画过渡
  • ✅ 键盘导航支持
  • ✅ 可访问性优化
