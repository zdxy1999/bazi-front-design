## 9. 使用规范

### 9.1 组件命名规范
- 基础组件：`bz-{组件名}` (如 `bz-button`)
- 业务组件：`bz-{业务}-{组件名}` (如 `bz-destiny-card`)

### 9.2 文件组织
```
components/
  ├── base/              # 基础组件
  ├── business/          # 业务组件
  └── layout/            # 布局组件
styles/
  ├── theme-light.wxss   # 浅色主题
  ├── theme-dark.wxss    # 深色主题
  └── variables.wxss     # 设计变量
```

### 9.3 主题切换
- 使用 `data-theme` 属性切换主题
- CSS 变量自动应用对应主题颜色
- 主题偏好保存在本地存储
