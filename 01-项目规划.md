# 八字命理小程序 - 项目规划

## 核心功能
1. **测算八字命盘** - 输入出生信息，生成八字命盘
2. **查看流年** - 查看每年的运势情况
3. **主题切换** - 浅色/深色两种主题

## 页面结构

### 1. 首页
- 入口页面
- 快速测算入口
- 历史记录入口
- 主题切换入口

### 2. 八字测算页
- 出生信息输入表单
- 测算按钮
- 测算结果展示

### 3. 命盘详情页
- 八字命盘可视化展示
- 五行分析
- 十神分析
- 详细解读

### 4. 流年运势页
- 年份选择器
- 流年运势展示
- 逐月运势查看

### 5. 历史记录页
- 历史测算记录列表
- 快速查看功能

### 6. 个人中心
- 用户信息
- 设置入口
- 主题切换

## 五行色彩体系

### 五行对应颜色
```css
/* 金 - Metal */
--color-metal-primary: #D4AF37;     /* 金色 */
--color-metal-light: #F4E5B3;       /* 浅金 */
--color-metal-dark: #AA8C2C;        /* 深金 */

/* 木 - Wood */
--color-wood-primary: #2ECC71;      /* 青绿 */
--color-wood-light: #58D68D;        /* 浅绿 */
--color-wood-dark: #27AE60;         /* 深绿 */

/* 水 - Water */
--color-water-primary: #3498DB;     /* 蓝色 */
--color-water-light: #5DADE2;       /* 浅蓝 */
--color-water-dark: #2980B9;        /* 深蓝 */

/* 火 - Fire */
--color-fire-primary: #E74C3C;      /* 红色 */
--color-fire-light: #EC7063;        /* 浅红 */
--color-fire-dark: #CB4335;         /* 深红 */

/* 土 - Earth */
--color-earth-primary: #F39C12;     /* 黄色 */
--color-earth-light: #F7DC6F;       /* 浅黄 */
--color-earth-dark: #D68910;        /* 深黄 */
```

## 流光特效说明
- **位置**：背景边缘内发光
- **效果**：五行颜色流动渐变
- **实现**：使用 CSS 渐变动画
- **颜色顺序**：金 → 木 → 水 → 火 → 土 → 金（循环）
